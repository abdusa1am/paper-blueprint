# Style Analysis

**Target length:** 180-230 lines unless the user explicitly asks for a different level of detail.

Based on the reference papers provided by the user.

**Goal:** extract reusable guidance for structure, visuals, formulas, citations, language, and layout so later writing sessions can stay stylistically consistent.

## Quantitative Style Snapshot

- Approximate page length
- Approximate section / subsection count
- Approximate figure count
- Approximate table count
- Approximate formula count
- Approximate citation count or citation density
- Approximate total references cited / bibliography size
- Overall implication for the target manuscript

## Overall Structure and Argument Progression

- content organization
- section hierarchy
- logical flow
- argument evolution from opening motivation to final synthesis
- overall writing style and tense patterns

## Figures

- total figure count
- figure-heavy sections
- figure-light or figure-free sections
- typical placement rhythm
- why figures are used overall

### Section-by-Section Figure Usage

For each major section, record:
- approximate figure count
- where figures appear (early / mid / late / after which subsection)
- why they appear there
- any recurring layout tendencies (multi-panel figures, dense legends, long captions, full-width placement, etc.)

## Tables

- total table count
- table-heavy sections
- table-light or table-free sections
- typical placement rhythm
- why tables are used overall

### Section-by-Section Table Usage

For each major section, record:
- approximate table count
- where tables appear
- why they appear there
- any recurring layout tendencies (wide tables, rotated tables, dense numeric grids, caption style, etc.)

## Formulas

- total formula count
- formula-heavy sections
- formula-light or formula-free sections
- typical placement rhythm
- why formulas are used overall

### Section-by-Section Formula Usage

For each major section, record:
- approximate formula count
- where formulas appear
- why they appear there
- whether formulas are mainly definitional, methodological, analytic, or evaluative

## Citation Usage

- approximate citation count or density for the whole paper
- citation-dense sections
- citation-light sections
- typical citation pattern across the paper
- why citations are distributed this way

### Section-by-Section Citation Usage

For each major section, record:
- approximate citation count or range
- citation role (motivation, prior work, taxonomy support, benchmark support, synthesis, etc.)
- whether citations appear as dense clusters, isolated support, or sustained background coverage

## Language and Presentation Patterns

- sentence structure
- technical terminology and definition style
- hedging vs. assertiveness
- transition patterns
- caption style
- how claims are qualified

## Section-Specific Observations

Summarize how the reference paper handles sections such as:
- introduction
- related work / survey body / technical core
- experiments / evaluation / discussion
- conclusion

Focus on how visuals, formulas, and citations support the rhetorical job of each section.

## Layout Tendencies to Reuse or Avoid

Note patterns such as:
- wide tables
- float density
- common figure/table placement choices
- caption length
- use of multi-panel figures
- whitespace tendencies
- whether the layout feels tight, airy, or crowded
